# Create an Integration Project

This quick start guide will help you create your first integration project using WSO2 Integrator: ICP.

## Prerequisites

- Installed **WSO2 Integrator: ICP** server. If you haven't installed it yet, please refer to the [Quick Start Guide]({{base_path}}/get-started/quick-start-guide/) to set up the ICP server.

## Step 1: Create an Integration Project
1. Navigate to the ICP dashboard at `https://localhost:9445/`.
2. Log in using your credentials. Default username and password are `admin`.
3. In the home page, click on the **`Create Project`** button.
4. Fill in the project details as below, and click on **`Create`** button.

| **Field**       | Value                | **Description**                                                             |
|------------------|----------------------|-----------------------------------------------------------------------------|
| **`Display Name`**     | My ICP Project       | The name that will be displayed for the project in the ICP dashboard.       |
| **`Description`**      | My first ICP project | A brief description of the project to help identify its purpose or details. |

<a href="{{base_path}}/assets/img/get-started/create-project/create_project.png"><img src="{{base_path}}/assets/img/get-started/create-project/create_project.png" alt="Create Project" width="70%"></a>  
5. You will be redirected to the project dashboard where you can start adding integrations.

## Step 2: Create an Integration
1. In the project dashboard, click on the **`+ Create`** button.
<a href="{{base_path}}/assets/img/get-started/create-project/project_home.png"><img src="{{base_path}}/assets/img/get-started/create-project/project_home.png" alt="Add Integration" width="70%"></a>
2. Fill in the integration details as below, and click on **`Create`** button.

| **Field**       | Value                                | **Description**                                                             |
|------------------|--------------------------------------|-----------------------------------------------------------------------------|
| **`Integration Name`** | My First BI Integration              | The name of the integration you are creating.                               |
| **`Description`**      | Simple Ballerina service integration | A brief description of the integration to help identify its purpose or details. |
| **`Type`**             | Ballerina Integration                | Select the type of integration you want to create.                           |

<a href="{{base_path}}/assets/img/get-started/create-project/create_integration.png"><img src="{{base_path}}/assets/img/get-started/create-project/create_integration.png" alt="Create Integration" width="70%"></a>    
3. You will be redirected to the integration details page where you can start connecting your integration.
